Import Process

The import process connects to your source platform, retrieves data, and creates corresponding records in HuduGlue.

Dry Run

When dry run is enabled, the import process will retrieve and process data but will not save anything to the database. This allows you to preview what will be imported.

Duplicate Prevention

Import mappings track source IDs to prevent duplicate imports. If you run the same import again, already-imported items will be skipped.
