Scheduler Status
Scheduler is installed and running! The systemd timer checks every minute and runs tasks based on their configured schedules.
Check Scheduler Status
sudo systemctl status itdocs-scheduler.timer
View Recent Scheduler Logs
sudo journalctl -u itdocs-scheduler.service -n 50 --no-pager
Manually Run Scheduler (for testing)
cd /home/administrator && ./venv/bin/python manage.py run_scheduler
Restart Scheduler
sudo systemctl restart itdocs-scheduler.timer
Production Deployment

For production environments, update paths in the service file:

# Edit service file
sudo nano /etc/systemd/system/itdocs-scheduler.service

# Update these paths:
WorkingDirectory=/opt/itdocs
ExecStart=/opt/itdocs/venv/bin/python manage.py run_scheduler
User=itdocs
Group=itdocs

# Reload and restart
sudo systemctl daemon-reload
sudo systemctl restart itdocs-scheduler.timer
